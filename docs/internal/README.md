# Internal Governance Documents

This directory contains internal governance records from the GrantLayer development
process. These are not intended for external developers.

**Contents:**

- Gate reports and go/no-go decisions
- Production readiness reviews
- Security audit artifacts
- Reviewer feedback triage records
- Public snapshot and release checklist records
- Pilot readiness and handoff packs

These documents are kept in the repository for historical completeness and traceability.
External developers looking for integration guidance should start at [docs/README.md](../README.md).
